1. **100% Архитектура Без Хардкода**: Весь текст, квесты, викторины и настройки распределены по структурированным JSON-базам данных (`content.json`, `config.json`).
2. **🔥 Вау-Фишка — «Путь Нурай в будущее»**: Комплексный игровой текстовый квест с ветвлением сюжета на базе Finite State Machine (FSM).
3. **🤖 Интеллектуальный ИИ-Ассистент (`/ask`)**: Система локального семантического поиска на основе регулярных выражений (`re`) и Keyword-анализа базы знаний портфолио.
4. **🏆 Интерактивная викторина**: Игра из 4 уровней со скорингом ответов и генерацией персональной мотивационной карты лидера.
5. **🛡️ Устойчивость к сбоям**: Архитектура полностью изолирована глобальными блоками `try-except`, логированием событий в `logs/bot.log` и защитой от падений при некорректном вводе данных.

## 🛠️ Быстрый Запуск локально

1. Установите зависимости:
   pip install -r requirements.txt

2. Запустите бота, передав токен параметром:
   python bot.py 8780561569:AAEDy5KlsiktXEwGhzjcL6_uhHGeNxRHwZU

   Или в режиме отладки:
   python bot.py 8780561569:AAEDy5KlsiktXEwGhzjcL6_uhHGeNxRHwZU debug
